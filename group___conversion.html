template<class UnitFrom , class UnitTo , typename T = double>
static T units::convert ( const T &  value)
inlinestatic

converts a value from one type to another.

Converts a value of a built-in arithmetic type to another unit. This does not change the type of value, only what it contains. E.g.

double result = convert<length::meters, length::feet>(1.0); // result == 3.28084
See also
unit_t for implicit conversion of unit containers.

template<typename T , typename Units , class = typename std::enable_if<std::is_arithmetic<T>::value && traits::is_unit_t<Units>::value>::type>
T units::unit_cast ( const Units &  value)
inline

Casts a unit container to an arithmetic type.

unit_cast can be used to remove the strong typing from a unit class, and convert it to a built-in arithmetic type. This may be useful for compatibility with libraries and legacy code that don't support unit_t types. E.g

meter_t unitVal(5);
double value = units::unit_cast<double>(unitVal); // value = 5.0
